 📰 AI-Based Fake News Detection Tool

 🤖 CodeVedX AI/ML Virtual Internship 2026 | Project 3

> **Can AI identify what is REAL and what is FAKE?**  
> This NLP-powered machine learning application analyzes news content and predicts whether it is **FAKE** or **REAL** — along with a confidence score.

🚀 Project Overview
In the digital age, misinformation spreads faster than ever.
This project uses **Natural Language Processing (NLP)** and **Machine Learning** to build an intelligent Fake News Detection Tool capable of analyzing news articles and classifying them as:
 🔴 **FAKE NEWS**  
 🟢 **REAL NEWS**

The system converts text into meaningful numerical features using **TF-IDF Vectorization** and performs classification using the **Passive Aggressive Classifier**.
 ✨ Key Features
| 🔢 | Feature | Description |
|---|---|---|
| 1️⃣ | 📰 Single News Detection | Analyze an individual news article |
| 2️⃣ | 🧠 AI Classification | Predict FAKE or REAL news |
| 3️⃣ | 📊 Confidence Score | Display prediction confidence |
| 4️⃣ | 🔄 Model Retraining | Train and save the ML model |
| 5️⃣ | 🧪 Batch Testing | Test 8 news samples simultaneously |
| 6️⃣ | 📈 Data Visualization | Generate 3 analysis charts |
