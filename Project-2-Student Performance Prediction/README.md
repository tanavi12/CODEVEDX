🎓 Student Performance Prediction System
Predict. Analyze. Improve. Succeed.

🤖 CodeVedX AI/ML Internship 2026 | Project 2
A machine learning-powered student analytics system that predicts a student's final academic performance using attendance, subject marks, study habits, sleep patterns, and extracurricular activities.
This project combines Exploratory Data Analysis (EDA), Data Visualization, and Machine Learning to transform student data into meaningful academic insights.

🌟 What This Project Does
📊 Analyzes student performance
🔍 Explores patterns and relationships in data
📈 Generates meaningful visualizations
🤖 Predicts final student grades using ML models
💡 Provides personalized improvement suggestions

🧠 How the System Works
Student Data
     ↓
Data Analysis & Cleaning
     ↓
Exploratory Data Analysis
     ↓
Data Visualization
     ↓
Machine Learning Models
     ↓
Final Grade Prediction
     ↓
Personalized Feedback 🎯

✨ Key Features
🔢| Feature
01| 📋 Dataset overview with shape, missing values, and sample records
02| 🔍 Exploratory Data Analysis and statistical insights
03| 📊 Four informative data visualizations
04| ➕ Add new student records to the dataset
05| 🤖 Train and compare Machine Learning models
06| 🎯 Predict final student grade
07| 💬 Generate personalized improvement suggestions

📚 Prediction Factors
The system predicts final performance using:
- 🏫 Attendance Percentage
- ➗ Mathematics Marks
- 🔬 Science Marks
- 📖 English Marks
- ⏱️ Study Hours per Day
- 😴 Sleep Hours
- 🎨 Extracurricular Activities


🤖 Machine Learning Models
📈 Linear Regression
Used to understand the relationship between student attributes and final academic performance.

🌲 Random Forest Regressor
An ensemble learning model used to capture complex patterns and improve prediction accuracy.

🏆 Model Performance
Model| MAE| R² Score
📈 Linear Regression| 0.16| 0.9998
🌲 Random Forest Regressor| 0.67| 0.9968
⭐ The models achieved excellent prediction performance on the available dataset.»


📊 Data Visualizations

Highlights relationships between different student performance factors.
